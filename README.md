# 🏛️ Tog-Wajaale City Administration - Land Registration System

 Overview
The **Tog-Wajaale City Administration Land Registration System** is a comprehensive, web-based solution designed to modernize and streamline land management processes in Tog-Wajaale City. This system replaces traditional paper-based methods with a secure, efficient, and user-friendly digital platform.

  Purpose
The system is built to serve the city administration, land registry officers, and citizens by providing:
-  Transparent land transaction processing
-  Secure deed document management
-  Automated revenue tracking and reporting
-  Efficient data storage and retrieval
-  Professional report generation

 Key Features

 Land Transaction Form
- Full owner information management
- Parcel ID and coordinates (UTM Zone 38N)
- Land category (Commercial/Residential/Mixed Use/Service)
- Land grade (A/B/C/D)
- Area calculation (L×W and Irregular methods)
- National ID photo upload (Front & Back)
- Owner photo upload

 Deed Document Registration
- Complete deed documentation
- Buyer information
- Transfer date tracking
- Tax and revenue management
- Professional deed number generation

 Restricted Deed Document
- Bank lending management
- Loan fee calculation (1%)
- Bank letter reference
- Title deed and identification numbers
- Annual tax clearance tracking

  Revenue & Reports
- Daily Revenue Summary - Track daily collections
- Monthly Revenue Summary - Complete monthly reports
- Quarterly Revenue Summary - Quarterly performance reports
- Excel Export** - Export all data to Excel format
- PDF Report Generation - Professional PDF reports
- CSV Export** - Data export for analysis

 Security Features
- User authentication with account creation
- Session management (4-hour timeout)
- **Rate limiting** - Maximum 5 saves per hour (prevents spam)
- **Duplicate prevention** - No duplicate records allowed
- Secure data storage in browser localStorage

 Accessibility
- Responsive Design** - Works on desktop, tablet, and mobile
- 24/7 Availability** - Access anytime, anywhere
- User-Friendly Interface** - Modern, intuitive design
- No Installation Required** - Works directly in browser
